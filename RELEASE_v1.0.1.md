# 微博逃生舱 v1.0.1

## 🎉 重大更新

### 核心功能

- ✨ **用户ID自动提取**：从登录页面URL智能识别并自动填充用户ID
- 🔐 **完整Cookie获取**：使用原生API获取所有Cookie（包括HttpOnly），从5个增加到12个
- 🔧 **线程异步优化**：修复Event Loop错误，实现正确的线程管理
- 📁 **配置路径优化**：使用用户可写目录，自动迁移旧配置

### UI/UX改进

- 🎨 **按钮优化**：Loading动画、智能状态管理、固定最小宽度、完善阴影效果
- 📊 **进度显示**：新增列表进度条，移除视频进度条，优化颜色透明度
- 📝 **友好日志**：中文提示、Emoji图标、减少刷屏
- 🎯 **智能交互**：按钮根据状态自动启用/禁用

### Bug修复

- 修复强制停止后按钮状态未更新
- 修复按钮阴影偶尔消失
- 修复Cookie不完整导致登录失败
- 修复用户ID未自动保存
- 修复应用重复打开问题
- 修复配置文件只读问题

## 🚀 快速开始

1. 下载并安装应用
2. 首次启动会自动创建配置文件
3. 进入「逃生设置」→「登录并自动获取Cookie」
4. 登录后访问个人主页，用户ID会自动提取
5. 返回「开始逃生」点击「开始」
